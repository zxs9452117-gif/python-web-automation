# 🚀 Python Web 自動化測試框架範例

這是一個基於 **Pytest + Playwright** 的自動化測試專案，展示了 POM 架構與 CI/CD 整合。

## 🛠 技術棧
- **語言**: Python 3.10+
- **框架**: Pytest
- **驅動**: Playwright (比 Selenium 更快、更穩定)
- **CI/CD**: GitHub Actions

## 📈 自動化流程
本專案整合了 GitHub Actions，每次提交程式碼都會自動執行測試並產出報告。

## 🚀 如何運行
1. `pip install -r requirements.txt`
2. `playwright install`
3. `pytest`
